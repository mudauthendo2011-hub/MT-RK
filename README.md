# Installation

1. Clone the repository:



git clone https://github.com/mudauthendo2011-hub/MT-RK.git

2. Install dependencies:



npm install

3. Configure your bot (add your WhatsApp session info, tokens, etc.)


4. Start the bot:



node index.js
